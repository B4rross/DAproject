\begin{DoxyNote}{Note}
The Edmonds-\/\+Karp algorithm is a special case of the Ford-\/\+Fulkerson algorithm. 

It uses Breadth-\/\+First Search to find the augmenting paths with the minimum number of edges 
\end{DoxyNote}
\begin{DoxyAttention}{Attention}
The time complexity of the Edmonds-\/\+Karp algorithm is O(\+V$\ast$\+E$^\wedge$2), where V is the number of vertices and E is the number of edges in the graph. 
\end{DoxyAttention}
